# Digital Clock ⏰

A simple and elegant digital clock built in Java using AWT/Swing. The application displays the current time and date in real-time with a clean UI.

---

## 🚀 Features

- ⏱️ Real-time clock display  
- 📅 Current date display  
- 🎨 Clean and minimal UI  
- 🔄 Auto-refreshing time  

---

## 🛠️ Tech Stack

- Java  
- AWT / Swing  

---

## ▶️ How to Run

```bash
# Compile
javac *.java

# Run
java main
